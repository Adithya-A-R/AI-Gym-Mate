# AI-Gym-Mate
AI GymMate is an AI-powered fitness and nutrition assistant that provides real-time exercise monitoring, posture correction, and personalized workout guidance using Computer Vision and Deep Learning.

The system uses MediaPipe Pose Estimation and a CNN-based exercise classification model to analyse human body movements through live camera input and uploaded workout videos.

🚀 Features
Real-time Exercise Monitoring
Posture Detection & Correction
CNN-based Exercise Classification
Repetition Counting
Calorie Estimation
Nutrition & Diet Tracking
Video Upload Analysis
Medical Condition–Based Exercise Filtering
Personalized Workout Recommendations

⚙️ System Workflow
Capture live workout or upload video
Detect body landmarks using MediaPipe
Calculate joint and ground angles
Feed angle data into CNN model
Classify exercise and evaluate posture
Generate feedback, rep count, and calorie estimation
